2. About Me – Highlights skills, experience, and goals
3. Projects Showcase – Interactive display of my best work with live demos
4. Contact Form – Easy way for visitors to connect
5. Responsive Design – Optimized for desktops, tablets, and smartphones
6. Smooth Navigation & Animations – Professional look and feel
